# Database-Task-1

## Installation



```bash
clone the project
cd to the project root

#make sure you have a postgres database,
configure your database credentials in the src/db.js file
#
Run npm install
Run npm run dev

Using postman/browser, navigate to localhost:3000,
check your database now.
    
```
